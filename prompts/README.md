# Prompts

This folder contains prompt templates and rules used for AI-assisted development and consistency.

## Structure

- **`_base/`** – Base rules and response standards applied across the project.
  - `base.rules.md` – Core rules and conventions.
  - `response.standard.md` – Expected response format and style.

## Usage

Reference these files when configuring Cursor rules, agents, or other AI tooling so that generated code and responses follow project standards.
